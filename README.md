# Chess Game Project ♟️

<img src="./frontend/public/chess_gif.gif"/>

This project is a comprehensive chess game web application, built with frontend architecture. The application provides a nice and neat interface for playing chess, with mode playing 2Players. The project is designed to demonstrate a robust and scalable architecture, using modern technologies (e.g. React, TypeScript, Vite).

## Tech Stack
* **Frontend**:
	+ React
	+ TypeScript
	+ Vite
	+ CSS

## Key Project Structure
