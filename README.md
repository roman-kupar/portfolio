# Chess Game Project ♟️

<img src="./frontend/public/chess_gif.gif"/>

This project is a comprehensive chess game web application, built with a combination of frontend and backend architecture. The application provides a nice and neat interface for playing chess, with modes playing 2Players or "AI" opponent, powered by Minimax algorithm. The project is designed to demonstrate a robust and scalable architecture, using modern technologies, different languages and services (e.g. React, Flask, C++).

## Tech Stack
* **Frontend**:
	+ React
	+ TypeScript
	+ Vite
	+ CSS
* **Backend**:
	+ Python
	+ Flask
	+ Chess Engine (built with C++)
